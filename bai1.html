What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage

With CSS, you can control the color,font,the size of text,the spacing between elements,how elements are positioned and laid out,what
backgroud images or backgroud colors are to be used ,different devices and screen sizes,and much more!

Tip: the word cascading means that style applied to a parent element will also apply to all children elements within the paren.So,if you
set the color of the body text to "blue",all headings,paragraphs,and other text element within the body will also get the same color(unless
you specify something else)!
